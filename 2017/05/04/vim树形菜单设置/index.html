前戏

linux下的vim编辑器是一个好东西，但是如何将它打造成一款编辑的神奇，更加有效的帮助我们进行开发呢，下面来介绍两种方法，让你飞起来

2. 部署安装

NerdTree使用

1、在linux命令行界面，输入vim

2、输入 :NERDTree ，回车

3、进入当前目录的树形界面，通过小键盘上下键，能移动选中的目录或文件

4、目录前面有+号，摁Enter会展开目录，文件前面是-号，摁Enter会在右侧窗口展现该文件的内容，并光标的焦点focus右侧。

5、ctr+w+h 光标focus左侧树形目录，ctrl+w+l 光标focus右侧文件显示窗口。多次摁 ctrl+w，光标自动在左右侧窗口切换

6、光标focus左侧树形窗口，摁? 弹出NERDTree的帮助，再次摁？关闭帮助显示

7、输入:q回车，关闭光标所在窗口

vim的设置.vimrc的设置：

set number
" 设置tab的宽度和使用空格替代tab
set shiftwidth=4
set tabstop=4
set expandtab
set softtabstop=4
map <F2> :retab <CR> :wq! <CR>
" 设置编码和文件编码
set enc=utf-8
set fenc=utf-8
"  搜索结果高亮
set hlsearch
" 语法高亮
syntax on
" 设置文件浏览器的当前目录
set bsdir=buffer
set autoindent
set cindent
set nopaste
set wrap
if $TERM !="linux"
  set mouse=a 
endif
" 把F8 映射到NERDTree插件
map <F8> :NERDTree<CR>
