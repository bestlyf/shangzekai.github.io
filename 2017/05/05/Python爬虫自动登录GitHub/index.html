前言

学习完了requests和BeautifulSoup的用法后，我们来看一下怎么样通过这两个工具来登录一些我们经常上的网站。接下来，我们以登录github来讲解一下python爬虫的自动登录

第一次请求

首先，我们想要模拟浏览器登录github这个网站的话，先要看一下，浏览器在登录这些网站的时候，是向哪个URL发送请求，并且发送什么样的参数,我们可以调出控制台，查看网络请求，如下图所示：

我们可以看到，登录的URL是：http://github.com/session,并且POST的数据为form Data中包含的数据，所以我们只需要使用requests的post方法，将这些Form Data的数据发送给特定的URL即可。
