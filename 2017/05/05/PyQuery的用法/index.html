前戏

之前的文章中，我们学习了正则表达式，晦涩难懂，但是可以满足我们的需求，接下来，我们继续学习了Beautiful Soup的用法，感觉双手可以解放了，但是我们还是不满足，思考着能不能一步就高潮

嗯，那么这次学习的内容就会能直接让大家高潮，因为PyQuery 来了，乍听名字，你一定联想到了 jQuery，如果你对 jQuery 熟悉，那么 PyQuery 来解析文档就是不二之选

PyQuery 是 Python 仿照 jQuery 的严格实现。语法与 jQuery 几乎完全相同，所以不用再去费心去记一些奇怪的方法了

安装

还是熟悉的味道，还是熟悉的配方
pip3 install pyquery

初始化

1. 直接html字符串

pq 参数可以直接传入 HTML 代码，doc 现在就相当于 jQuery 里面的 $ 符号了

2. 直接传入URL

这里就像直接请求了一个网页一样，类似用 urllib 来直接请求这个链接，得到 HTML 代码

3. 传文件

可以直接传入文件路径

4. lxml.etree

可以首先用 lxml 的 etree 处理一下代码，这样如果你的 HTML 代码出现一些不完整或者疏漏，都会自动转化为完整清晰结构的 HTML代码
